## Modern React Deep Dive, 2022, Wikibooks, Kim Yong-chan

As the saying goes, the younger generation is to be held in awe. Even back when I was utterly obsessed with development, it never occurred to me to open up the source code of a DBMS or to crack open Angular's code and analyze it. The only time I ever rummaged through a Java decompiler was when I ran into some error that didn't even show up on Stack Overflow, with the mindset that if this didn't solve it, I'd quit development for good. The fact that a book has come out that dissects React to this level is astonishing. What's even more astonishing is that it was written by a developer with far fewer years of experience than me. I think it's fair to take this as a sign of just how high the development standards of Korean-speaking developers have risen.
