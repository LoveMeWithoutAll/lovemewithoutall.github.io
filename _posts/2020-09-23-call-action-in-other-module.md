`dispatch`를 쓰면 된다.

```javascript
const callActionInOtherModule = async (store, { payload }) => {
  try {
    store.dispatch("moduleB/actionName", { payload }, { root: true });
  } catch (e) {
    console.error(e);
  }
};
```

`{ root: true }`를 꼭 넣어야 한다.
